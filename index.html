Client Resource Management System - Philadelphia ReCast Program

Worked in a 4-person team to deliver a complete client resource management system for the Philadelphia ReCast Program. Completed the analysis phase of the project by conducting interviews with the client, understanding and documenting their business requirements and system goals. Utilized an Object-Oriented Software Development Lifecycle approach which included Unified Modeling Language (UML) tools such as Activity Diagrams, Use Cases, and User Stories.

Andriod App - Audio Books

Created an app in Andriod Studios. This app utilized components to create a service to allow the user to select and play audio books of their chosing. The app read from an API to grab book information. If a book was never played before, the app would stream the books audio and download the book's mp3 file in the background. If the book was previously played, the app would simply play the downloaded file instead of streaming. You could play, pause, and stop the audio. You can also use a seek-bar to go change the progress of the book. If the user closed the app the audio would continue to play in the background.

Video Game - Jeopardy!

Created the game of Jeopardy! using C# and databases. I stored all player, question, and answer information in a database. The application I created follows the official rules of the real game with the exception that we only allow 2 players. Two players take turns answering questions found in different subject categories for various points. Some questions have have more points than others. If a player answers a question incorrectly they will loss the question points and the second player gets a chance to answer. We included the second round and final jeopardy round as well.
